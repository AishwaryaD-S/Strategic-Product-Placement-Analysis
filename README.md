# Strategic Product Placement & Sales Performance Analysis
Developed for SmartBridge SkillBridge Project.

## 🚀 Project Overview
This project provides a data-driven solution for retail managers to optimize shelf placement and pricing strategies. It features a Flask-based web interface integrated with a professional Tableau dashboard.

## 🛠️ Technology Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML5, CSS3, Bootstrap
- **Analytics:** Tableau Desktop
- **Documentation:** Microsoft Word/PDF

## 📂 Project Structure
- **/web-app**: Contains the Python server and HTML templates.
- **/tableau-project**: Includes the workbook and retail dataset.
- **/documentation**: Contains all 12 mandatory project report documents.
